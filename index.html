Desarrollo Web

Contacta con nosotros si quieres desarrollar una aplicación completa, un módulo de tu aplicación, una integración con un servicio externo o mejorar el rendimiento y la escalabilidad de tu aplicación.

Desarrollar una aplicación o módulo

Desarrollamos el front-end rico usando Angular o React. En el back-end exponemos una API programada con Symfony o NodeJS.

Mediante una comunicación estrecha buscamos llegar a la raíz del problema para encontrar las soluciones más simples para obtener unos resultados más satisfactorios.

Trabajamos en sprints de dos semanas. Al principio de cada sprint nos centramos en definir qué es lo que más valor aporta al siguiente desarrollo.

Garantizamos que la aplicación se comporta como se espera mediante tests automáticos que reducen los costes de mantenimiento. De esta forma nos aseguramos de que se puede añadir funcionalidad sin miedo a que deje de funcionar lo que ya se ha entregado.

Desarrollo de integraciones

Servicios de terceros: Google, Dropbox, Slack, Twitter, Amazon...

Seguridad: Single sign-on, OAuth, LDAP, Active Directory y ACL


Mejorar el rendimiento

Análisis en profundidad para detectar de cuellos de botella

Optimización de queries, índices y tablas en bases de datos

Acelerar el acceso a datos mediante cachés

Prevenir incidencias mediante la correcta monitorización del sistema

Realizar las tareas más costosas de forma asíncrona utilizando colas
